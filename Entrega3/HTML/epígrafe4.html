Pie, stars, persp, image y contour

Pie

La función pie permite crear un diagrama de sectores de los datos suministrados.


Leemos por ejemplo los datos proporcionados en el fichero datos.txt. Lanzamos un diagrama de sectores usando la variable sexo: 

datos<-read.csv("Datos.txt", header=TRUE, sep=",")
pie(table(datos$Sexo), col=c("black","orange"), main="Sexo")

Creamos un vector con ciudades de Andalucía donde residen estudiantes que cursan la asignatura. Llamamos a la función pie para mostrar el diagrama de sectores asociado.

Provincia<-c("GR","CO","CO","AL","AL","GR","JA","GR","CA","SE","JA","CA","GR","GR", "GR")
pie(table(Provincia), col=c("green", "red", "blue", "orange", "black","yellow"))

Volviendo al dataset de Iris, mostramos un último diagrama de sectores para mostrar la proporción de especies:

mytable <- table(iris$Species)
lbls <- paste(names(mytable), "\n", mytable, sep="")
pie(mytable, labels = lbls, main="Especies de Iris")

Stars

La función stars realiza un diagrama de estrellas, una por individuo, con información de todas las variables.


Mostramos diferentes ejemplos que se han realizado usando esta función:

datos<-read.csv("Datos.txt", header=TRUE, sep=",")
Peso<-datos$Peso
Altura<-datos$Altura
Edad<-datos$Edad
datos2<-matrix(c(Peso,Altura,Edad),ncol=3)

stars(matrix(c(Peso,Altura,Edad),ncol=3),main="Diagrama de estrellas")

stars(matrix(c(Peso,Altura,Edad),ncol=3),main="Diagrama de estrellas", draw.segments=TRUE)

Usamos el dataset mtcars: 

stars(mtcars[, 1:7], key.loc = c(14, 2), main = "Motor Trend Cars", full = FALSE)

palette(rainbow(12, s = 0.6, v = 0.75))
stars(mtcars[, 1:7], len = 0.8, key.loc = c(12, 1.5), main = "Motor Trend Cars", draw.segments = TRUE)

Y, por último, el dataset USJudgeRatings: 

data(USJudgeRatings)
stars(USJudgeRatings, labels = abbreviate(case.names(USJudgeRatings)),
                key.loc = c(13, 1.5), main = "Judge not ...", len = 0.8)

Persp

La función persp crea una gráfico en perspectiva.


Definimos una función y la llamamos después de realizar el producto exterior (función outer). 

x <- seq(-10, 10, length= 30)
y <- x
f <- function(x, y) { r <- sqrt(x^2+y^2); 10 * sin(r)/r }
z <- outer(x, y, f)
z[is.na(z)] <- 1
op <- par(bg = "white")
persp(x, y, z, theta = 30, phi = 30, expand = 0.5, col = c("black","white"))

Definimos una nueva función y le aplicamos el producto exterior con x e y. Acto seguido, llamamos a la función persp para mostrar el gráfico en perspectiva.

x <- y <- seq(-10, 10, length = 50);

z <- outer(x, y, function(x,y) {
               r <- sqrt(x^2 + y^2)+3;
               sin(r)/r});

persp(x, y, z, theta=15, phi=40, expand=1, border='#333333', col=c("black", "white"), shade=0.2);

Mostramos otro ejemplo:

x <- seq(-10, 10, length=40)
y <- seq(-10, 10, length=40)

z <- matrix (
                outer(x, y, function(x, y) {
                14 - sqrt( (x+cos(y)/10) ^2 + (y - sin(x)/5) ^2) +
        cos((x-2)/3)*sin(x) + cos((y-2)/4)*cos(y) + sqrt( (x+3) ^ 2 + (y/2-2) ^2)
                }),
                nrow=length(x)
)

persp(x, y, z, col=c("black","white"))

Contour

La función contour realiza un mapa de nivel.


Mostramos diferentes ejemplos y funcionalidades de esta función:

x <- y <- seq(-4*pi, 4*pi, len = 27)
r <- sqrt(outer(x^2, y^2, "+"))
opar <- par(mfrow = c(2, 2), mar = rep(0, 4))
for(f in pi^(0:3))
        contour(cos(r^2)*exp(-r/f), drawlabels = FALSE, axes = FALSE, frame = TRUE)

Destacamos su uso combinado con image que explicaremos a continuación.

Image

La función image dibuja una representación utilizando un código de color o una escala de grises.

x <- y <- seq(-4*pi, 4*pi, length.out = 27)
r <- sqrt(outer(x^2, y^2, "+"))
image(z = z <- cos(r^2)*exp(-r/6), col = gray.colors(33))

image(z, axes = FALSE, main = "Math can be beautiful ...", xlab = expression(cos(r^2) * e^{-r/6}))
contour(z, add = TRUE, drawlabels = FALSE)

Mostramos en un mapa de calor la representación de un volcán, del dataset volcano.

image(t(volcano)[ncol(volcano):1,])

x <- 10*(1:nrow(volcano))
y <- 10*(1:ncol(volcano))
image(x, y, volcano, col = hcl.colors(100, "terrain"), axes = FALSE)
contour(x, y, volcano, levels = seq(90, 200, by = 5), add = TRUE, col = "brown")
axis(1, at = seq(100, 800, by = 100))
axis(2, at = seq(100, 600, by = 100))
box()
title(main = "Maunga Whau Volcano", font.main = 4)
