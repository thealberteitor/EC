Introducción

En este proyecto final de la asignatura de Estadística Computacional se realizarán una serie de ejercicios con el lenguaje de programación R. Como restricción, el único libro disponible será knitr, usado para la generación automática de este archivo HTML. Es decir, el resto de funciones o módulos tendrán que ser implementados por el alumno.

La sentencia para añadir el libro knitr es la siguiente:

library(knitr)

Y, para generar el archivo HTML, bastaría con dirigirnos a nuestro directorio de trabajo e introducir el comando knit("1.Rhtml")
