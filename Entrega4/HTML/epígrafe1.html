Función de densidad y distribución

Distribución Normal

dnorm, pnorm

Las funciones dnorm y pnorm sirven para representar la función de densidad y de distribución de una distribución Normal

Hacemos uso de la función Dibuja del guión de prácticas.

Dibuja_normal<-function(n=30,alfa=.001,fil=1,col=1){
        #Dibuja la funcion de densidad
        #y la de distribucion de la normal
        n<-as.integer(n)
        if (n<10) stop("Hacen falta mas numeros")
        a<-qnorm(alfa)
        b<-qnorm(1-alfa)

        par(mfrow=c(fil,col))
        for (i in 1:(fil*col)){
                x<-seq(a,b,,i*n)
                plot(x,dnorm(x), col="red")
                title(main="Función de densidad", ylab="y", col="black")
        }


        par(mfrow=c(fil,col))
        for (i in 1:(fil*col)){
                x<-seq(a,b,,i*n)
                plot(x,pnorm(x), col="blue")
                title(main="Función de distribución", ylab="y", col="black")
        }

}

Llamamos a la función anterior: 

Dibuja_normal(n=50, alfa=.001, fil=2, col=1)

Distribución Uniforme

dunif, punif


Modificamos la función anterior Dibuja para representar la distribución uniforme entre 0 y 1. Para ello añadimos dos nuevos parámetros que se le pasarán a dunif y a punif cuando se vayan a lanzar con plot

Dibuja_uniforme<-function(n=30,alfa=.001,fil=1,col=1, min1=0, max1=1){
        #Dibuja la funcion de densidad
        #y la de distribucion de la uniforme
        n<-as.integer(n)
        if (n<10) stop("Hacen falta mas numeros")
        a<-qnorm(alfa)
        b<-qnorm(1-alfa)

        par(mfrow=c(fil,col))
        for (i in 1:(fil*col)){
                x<-seq(a,b,,i*n)
                plot(x,dunif(x, min=min1, max=max1), ylab="y", col="red")
                title(main="Función de densidad", col="black")

        }
        par(mfrow=c(fil,col))
        for (i in 1:(fil*col)){
                x<-seq(a,b,,i*n)
                plot(x,punif(x, min=min1, max=max1), ylab="y", col="blue")
                title(main="Función de distribución", col="black")
        }

}

Llamamos a la función anterior para representar las funciones de densidad y uniforme de la distribución uniforme: 

Dibuja_uniforme(n=50, alfa=.001, fil=1, col=1, min1=0, max1=1)

Distribución de Poisson

Escogemos la distribución de Poisson, por ejemplo, para mostrar sus gráficas con la ayuda de dos funciones:

dpois, ppois

Modificamos la función para representar la función de densidad y distribución de la Poisson.

Le pasamos el valor lambda por parámetro

Dibuja_poisson<-function(n=30,alfa=.001, fil=1, col=1, l=2){
        #Dibuja la funcion de densidad
        #y la de distribucion de Poisson
        n<-as.integer(n)
        if (n<10) stop("Hacen falta mas numeros")
        a<-qnorm(alfa)
        b<-qnorm(1-alfa)

        par(mfrow=c(fil,col))
        for (i in 1:(fil*col)){
                x<-seq(0,n)
                plot(x,dpois(x, lambda=l), ylab="y", col="red")
                title(main="Función de densidad", col="black")

        }
        par(mfrow=c(fil,col))
        for (i in 1:(fil*col)){
                x<-seq(0,n)
                plot(x,ppois(x, lambda=l), ylab="y", col="blue")
                title(main="Función de distribución", col="black")
        }

}

Dibuja_poisson(n=60, l=2)
